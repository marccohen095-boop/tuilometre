# Tuilomètre

Calcule les m² d'une toiture sans grimper : choisis la tuile (photo), compte les tuiles en horizontal et en vertical depuis le sol, l'appli sort la surface.

- 10 couvertures les plus posées en Belgique (terre cuite, béton, ardoises) avec cotes fabricant
- Le pureau / lattage / emboîtement est pris en compte automatiquement
- Additionne plusieurs pans pour le total toiture
- Bouton 📷 pour remplacer une photo par la tienne
- Fonctionne hors ligne, sur téléphone et PC

## Utilisation sur Android

Ouvre la page du site (GitHub Pages) dans Chrome → menu ⋮ → « Ajouter à l'écran d'accueil ».
